## Номер варианта
**Вариант 3: XOR-шифрование**  
Реализация побитового шифрования XOR с ключом для строк (UTF-8) и бинарных данных.  
Ключ используется циклически. Приложение предоставляет графический интерфейс для шифрования и дешифрования текстовых и бинарных (Base64) данных.

В процессе разработки применялись следующие инструменты отладки:

## Выводы по итогам комплексного тестирования
В ходе выполнения работы были применены следующие виды и техники тестирования:

Разработка через тестирование (TDD) – автоматизированные модульные тесты написаны до реализации методов.

Модульное тестирование – покрытие всех публичных методов шифрования/дешифрования позитивными и негативными тестами.

Дымовое тестирование (Smoke test) – быстрая проверка основных функций после внесения исправлений.

Ручное нефункциональное тестирование интерфейса и валидации.

Отладка с точками останова для верификации внутренней логики и циклического применения ключа.
